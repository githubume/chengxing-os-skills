# 成行 OS 专业服务

开源 Skill 让任何人先体验方法；专业服务处理需要人工判断、持续复盘或机构适配的复杂问题。

## 服务方向

| 服务 | 适合对象 | 典型交付 |
|---|---|---|
| **B2 复杂行为系统梳理** | 反复尝试失败、多人相互影响、变量较多的个人或家庭 | 行为循环图、竞争假设、最小实验、指标和复盘框架 |
| **Skill 与知识库定制** | 学校、教育机构、企业和专业团队 | 场景清单、工作流、风险边界、本地知识库与测试用例 |
| **成长行为工作坊** | 家长、教师、管理者和项目团队 | 场景行为分析、案例演练、行动设计和复盘工具 |
| **AI 本地部署与培训** | 希望使用 Claude Code、Codex 的组织 | 安装配置、内部方法接入、隐私边界和使用培训 |
| **持续跟踪与方法复盘** | 需要 7/14/30 天人工回看的人或团队 | 执行记录、效果与副作用分离、方案迭代建议 |

具体周期和交付范围在了解目标后单独确认，不以固定套餐替代真实需求判断。

## 合作流程

1. **非敏感需求说明**：只描述目标、使用场景、人数和希望交付物；
2. **范围与安全判断**：明确能做、不能做、数据边界和专业转介条件；
3. **小范围验证**：先围绕一个真实场景完成方法或原型；
4. **交付与复盘**：提供可执行材料、使用说明和后续改进建议。

## 我们不提供

- 医疗、心理诊断或危机干预；
- 对行为改变、学习成绩或关系修复的结果保证；
- 秘密监控、儿童标签、不成比例惩罚或侵犯第三方隐私的方案；
- 用未经验证的专家、机构或合作名称为服务背书。
