# 成行 OS 专业服务

本仓库的三个开源 Skill 用于公开验证成长行为学的工作方法。可提供：

1. 个人、家庭或团队的行为工作流定制；
2. 学校、机构和企业的 Skill/知识库本地化；
3. 成长行为学与场景行为工程工作坊；
4. B2 复杂行为系统的方法梳理和人工复盘；
5. Claude Code、Codex 的本地部署与使用培训。

服务不包含医疗、心理诊断、危机干预，也不承诺具体行为或成绩结果。

请通过 [服务咨询 Issue](https://github.com/githubume/chengxing-os-skills/issues/new?template=service-inquiry.yml) 提交需求概况。只写目标、使用场景、人数和希望交付物，不要填写个人身份、儿童资料、健康记录、真实聊天或危机细节。
