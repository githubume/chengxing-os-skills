# 隐私说明

## 默认行为

- 不保存对话和画像。
- 不上传数据，不含遥测、云同步或自动消息。
- 画像、案例和导出均需先展示预览，再取得本次明确确认。

## 本地路径

经用户确认的数据保存在：

```text
~/.chengxing-os/
```

插件升级和兼容适配器卸载不会删除该目录。用户可以查看、导出或删除自己的本地文件。

## 数据最少化

- 亲子 Skill 只记录必要的年龄/年级段，不保存姓名、学校、班级、地址或联系方式。
- 冲突 Skill 不保存对方姓名、原始聊天、精确地点、私密影像或可能暴露求助计划的信息。
- 公开仓库只含虚构示例，不含真实用户案例。

请勿在 GitHub Issue 中提交可识别个人、儿童、健康或危机信息。
