# 反刍转行动

## 必须先读

读取 `../../references/chengxing-shared/safety/routing.md`、`../../references/chengxing-shared/core-models/behavior-method.md` 和 `../../references/chengxing-shared/quality-gates/output-quality.md`。自伤、他伤、危险驾驶、报复计划、跟踪或无法保持安全时直接 B3，不用专注技巧替代安全行动。

## 执行

1. 把用户记得的原话/动作、自己的解释和未知信息分栏；记不清就明确“不足以判断”。
2. 判断当前属于可立即行动、需要沟通、需补信息、暂不可控或需专业支持。
3. 最多保留 3 个非诊断假设，不反复证明对方动机。
4. 从以下选一个最小动作：写一句事实、整理一个请求、记录待核实问题、完成一个与冲突无关的恢复动作，或暂缓联系。
5. 动作时长由用户按状态选择；结束后用新信息更新判断，不把“想明白”设为开始条件。
6. 若准备发送长消息，先缩为一件事实、一个需要和一个请求，并在发送前检查安全与目的。

## 输出

- B1/B2 与安全说明；
- 事实、解释、未知、可控四栏；
- 一个当前动作及结束条件；
- 冲动联系/报复时的替代动作；
- 最多 3 个指标，例如反刍时长、返回日常任务所需时间、冲动联系次数；
- 恢复、第 3/7 天复盘和升级条件。

不得宣称停止思考就能治愈焦虑或抑郁，不把所有原因归为童年，不要求压下情绪，也不根据单次反刍诊断疾病或人格。
