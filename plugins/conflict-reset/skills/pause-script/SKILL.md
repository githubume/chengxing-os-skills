# 当场暂停脚本

## 必须先读

读取 `../../references/chengxing-shared/safety/routing.md`、`../../references/chengxing-shared/core-models/behavior-method.md` 和 `../../references/chengxing-shared/quality-gates/output-quality.md`。任何 B3 信号都停止普通脚本；暴力、胁迫或跟踪中不承诺返回现场、不告知可能施害者求助计划。

## 前置判断

确认：场景目前安全、用户能自主离开或暂停、对方没有阻止离开或报复威胁、返回谈话是自愿且可行。若信息不足，先问一个直接安全问题。

## 结构

```text
我注意到自己的[可观察升级信号]。
我现在停止[会造成伤害的动作]，需要暂停到[可兑现时间/条件]。
到时我会用[安全渠道]联系，只讨论[一件具体事情]。
```

1. 只描述自己的状态和动作，不诊断、指责或命令对方冷静。
2. 时间由用户按场景选择，不硬套固定分钟数；无法承诺具体时间时给重新联系条件。
3. 暂停期间不追发长消息、不隔门争辩、不借机惩罚或删除对方资源。
4. 返回前检查是否已降到可谈水平；没有降级则发送一次简短更新，不失联。

## 输出

- B1/B2 与安全前提；
- 一句现场版和一句文字版；
- 暂停时的一个行为动作；
- 返回时间/条件和讨论范围；
- 若未能按时返回的更新句；
- 失败恢复和升级条件。

暂停不是冷暴力。不得把无限沉默包装为边界，不承诺在不安全现场返回，不把暂停用作迫使道歉、服从或追逐的手段。
