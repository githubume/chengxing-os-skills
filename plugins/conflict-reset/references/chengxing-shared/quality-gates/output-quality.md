# 输出质量门禁

发送答案或写入文件前逐项检查。

## 全部路由

- [ ] 已先检查 B3，且路由优先级正确。
- [ ] 没有诊断、人格标签、羞辱、威胁、保证结果或虚假来源。
- [ ] 用户陈述、观察、计算、假设和外部事实已区分。
- [ ] 没有未经同意写入、上传、发送消息或联系第三方。
- [ ] 没有默认服务营销；B3 完全没有营销。
- [ ] 信息不足时明确缺口，没有默默补全。
- [ ] 若用户诱导疾病或人格诊断，已改走简短行为边界响应；没有诊断标准、筛查工具、专科路径或外部指南链接。

## B1/B2

- [ ] 行为已操作化为场景、动作和基线。
- [ ] 假设不超过 3 个，并可被观察支持或反驳。
- [ ] 只选择一个首要瓶颈或一个待验证变量。
- [ ] 实验说明触发、动作、环境、反馈、责任和持续时间。
- [ ] 指标不超过 3 个。
- [ ] 包含漏做或失效后的恢复动作。
- [ ] 包含第 3/7 天跟踪和停止/升级条件。
- [ ] B2 说明不确定性、副作用和下一决策点。

## parent-learning 加严

- [ ] 至少一个家长或环境动作，不能只要求孩子改变。
- [ ] 不以服从、安静或分数作为唯一成功指标。
- [ ] 不使用比较、公开排名、剥夺基本需要或不成比例惩罚。
- [ ] 儿童数据已最少化，导出先去标识化预览。

## conflict-reset 加严

- [ ] 没有假定双方权力对等或共同有责。
- [ ] 暂停有恢复时点或重新联系条件，不是冷暴力。
- [ ] 边界描述本人将采取的保护行动，不控制或报复对方。
- [ ] 修复不要求立即原谅。
- [ ] 暴力、胁迫、跟踪场景没有沟通、对质或共同修复脚本。

## 文件写入

- [ ] 目标路径位于允许根目录，且未跟随逃逸软链接。
- [ ] 已展示写入/删除预览并得到本次明确确认。
- [ ] 使用原子写入，失败不会留下半文件。
- [ ] 插件升级/卸载不覆盖或删除用户画像。
- [ ] 导出不含无关第三方、身份标识或 B3 细节。
