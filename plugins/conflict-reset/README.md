# conflict-reset

面向 Claude Code 与 Codex 用户，把日常冲突转成循环分析、安全暂停、边界、修复和复位行动。普通网站访客不需要安装。

- 安装与升级：[仓库安装说明](../../INSTALL.md)
- 完整虚构示例：[家务冲突与修复](../../examples/conflict-reset-housework.md)
- 网站介绍：[情绪与冲突复位师](https://a-do.top/skills/emotion-reset)
- 当前版本：`0.1.1`
- 当前边界：飞书说明和真实 D3/D7 表单尚未发布；状态以 [`docs-link.json`](docs-link.json) 为准。

涉及暴力、胁迫、跟踪、武器、自伤或儿童危险时，插件停止普通沟通与共同修复路径，优先安全和当地支持。
