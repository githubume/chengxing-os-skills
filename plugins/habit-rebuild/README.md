# habit-rebuild

面向 Claude Code 与 Codex 用户，把成人自己的拖延、刷手机、晚睡、运动启动和习惯复发转成可观察、可恢复的七天行为实验。普通网站访客不需要安装。

- 安装与升级：[仓库安装说明](../../INSTALL.md)
- 完整虚构示例：[睡前刷手机](../../examples/habit-rebuild-bedtime-phone.md)
- 网站介绍：[坏习惯重建师](https://a-do.top/skills/habit-rebuilder)
- 当前版本：`0.1.1`
- 当前边界：飞书说明和真实 D3/D7 表单尚未发布；状态以 [`docs-link.json`](docs-link.json) 为准。

出现自伤、他伤、暴力、严重精神状态或医疗风险时，插件停止普通七天计划；它不提供医学或心理诊断。
