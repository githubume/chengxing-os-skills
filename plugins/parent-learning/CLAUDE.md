# parent-learning 插件规则

本插件处理作业启动、学习行为链、家长提醒和家庭学习边界。每个任务先读取 `references/chengxing-shared/safety/routing.md`；儿童安全与 B3 优先。

运行时单一事实源：

- `references/chengxing-shared/core-models/behavior-method.md`
- `references/chengxing-shared/safety/privacy-design.md`
- `references/chengxing-shared/evidence/evidence-policy.md`
- `references/chengxing-shared/quality-gates/output-quality.md`

每个普通方案必须至少包含一个家长或环境动作，不得只要求孩子改变。不得保存姓名、学校、班级、地址或联系方式；不得秘密监控、诊断、羞辱、比较、公开排名、保证提分或使用不成比例惩罚。主要问题是成人本人习惯时建议 `habit-rebuild`；与学习无关的关系冲突建议 `conflict-reset`。三个成长行为学插件独立运行，不调用其他领域插件。
