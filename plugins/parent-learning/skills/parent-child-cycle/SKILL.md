# 亲子学习循环

## 必须先读

读取 `../../references/chengxing-shared/safety/routing.md`、`../../references/chengxing-shared/core-models/behavior-method.md` 和 `../../references/chengxing-shared/quality-gates/output-quality.md`。家庭暴力、伤害威胁、儿童危险或严重危机直接 B3；不得用沟通脚本覆盖安全问题。

## 还原循环

按真实顺序记录：

```text
任务/状态 → 家长第一次动作 → 孩子回应 → 家长升级
→ 孩子下一回应 → 双方即时结果 → 下次预期
```

不推测孩子或家长的内心为事实。成人拥有更多权力和调节责任，不能把成人情绪管理交给孩子。

## 执行

1. 询问最近一次具体事件、提醒次数、音量/动作变化和最终如何结束。
2. 分别识别孩子与家长行为的即时功能和长期代价。
3. 找出最早可介入点，优先在冲突升级前改变提示、等待、任务结构或环境。
4. 定义孩子一个可观察动作、家长一个可观察动作和环境一个支持动作。
5. 边界提前约定，不在冲突中临时加码。
6. 若双方说法冲突，标记“单方陈述/待核实”，不判定人格或动机。

## 输出

- B1/B2 路由与最近一次循环图；
- 双方即时收益和长期代价；
- 最早升级点；
- 孩子、家长、环境各一个改变；
- 一句低刺激提醒和一句家长自我暂停语；
- 最多 3 个指标；
- 漏做、冲突后的恢复步骤；
- 停止和专业升级条件。

不得要求孩子立即原谅、安抚家长或长期承担家庭管理责任。
