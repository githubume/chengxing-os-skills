# 家庭手机边界

## 必须先读

读取 `../../references/chengxing-shared/safety/routing.md`、`../../references/chengxing-shared/core-models/behavior-method.md` 和 `../../references/chengxing-shared/quality-gates/output-quality.md`。网络威胁、勒索私密影像、成人诱骗、自伤、暴力或儿童安全风险直接 B3。

## 分析

1. 明确设备、场景、时间段、当前规则、实际使用和最近一次冲突。
2. 区分娱乐、社交、学习、联络安全和情绪调节功能，不把所有屏幕时间混为一类。
3. 检查家长规则是否一致、成人是否示范、设备是否随手可得、替代活动是否真实可用。
4. 选择一个首轮边界，不突然叠加多项限制。

## 设计边界

边界至少包含：

- 适用对象：规则是否也约束成人；
- 允许场景与禁止场景；
- 开始/结束提示和固定存放位置；
- 学习、联络与紧急使用例外；
- 孩子可以提出修改的时间和方式；
- 越界后的可预期、成比例处理；
- 线下替代活动；
- 冲突后的恢复流程。

## 禁止

不提供秘密监控、破解账号、诱骗密码、公开羞辱、无限期没收、剥夺睡眠/饮食/教育/必要联系，或以恐惧让孩子服从。不得把一次发脾气直接诊断为网络成瘾。

## 输出

- B1/B2 路由与手机行为功能假设；
- 当前规则失效循环；
- 一页家庭媒体协议草案；
- 孩子、家长和环境各一个动作；
- 最多 3 个过程指标；
- 越界恢复和第 3/7 天复盘；
- B3 与专业升级条件。
