# parent-learning

面向 Claude Code 与 Codex 用户，分析作业启动、反复催促、手机边界和亲子学习循环，让孩子、家长与环境共同参与改变。普通网站访客不需要安装。

- 安装与升级：[仓库安装说明](../../INSTALL.md)
- 完整虚构示例：[作业启动与催促](../../examples/parent-learning-homework-start.md)
- 网站介绍：[亲子学习行为教练](https://a-do.top/skills/parent-child-learning)
- 当前版本：`0.1.1`
- 当前边界：飞书说明和真实 D3/D7 表单尚未发布；状态以 [`docs-link.json`](docs-link.json) 为准。

插件不要求儿童真实姓名、学校、班级、照片、语音或精确地址，也不把改变责任只压给儿童。
