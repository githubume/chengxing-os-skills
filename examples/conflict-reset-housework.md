# conflict-reset 示例：家务冲突循环与暂停修复

本示例完全虚构，不来自真实个人或关系。

## 用户请求

> 我和伴侣每周都会因为厨房收尾争吵。对方问“怎么又没收”，我会解释工作很累；对方继续追问，我提高音量说“你只会挑错”，随后对方沉默离开。通常第二天才恢复，但事情没有重新约定。没有威胁、暴力、限制离开或安全风险。我想先改掉自己的升级动作，又不想把所有责任都揽过来。

## 预期触发

- 插件：`conflict-reset`
- 工作流：`cycle-analysis`，随后 `pause-script` 和 `repair-conversation`
- 路由：B2；重复发生且双方行为相互强化，但当前没有 B3 信息。

## 事实、解释与未知

| 类型 | 内容 |
|---|---|
| 可核查事实 | 厨房未按原约定收尾；对方提问；用户解释后提高音量并说伤人话；对方沉默离开；第二天恢复日常 |
| 用户解释 | “对方只会挑错” |
| 未知 | 对方当时的意图、感受和沉默功能；双方对厨房收尾标准是否一致 |

## 最近一次循环

```text
工作疲惫＋厨房未收尾
→ 对方提问
→ 用户解释为指责
→ 辩解、提高音量和反击
→ 对方沉默离开
→ 当下停止追问、双方获得距离
→ 规则仍未澄清，下周重复
```

这不是双方“同责”的裁决，只是基于用户单方陈述还原可观察顺序。首轮只改变用户可控制的最早升级点：注意到语速或音量上升后，停止反击并明确暂停。

## 暂停脚本

> 我注意到自己开始提高音量，再说下去会伤人。我先暂停到今晚九点；九点我会回来，只谈厨房收尾的具体分工。如果需要改时间，我会在九点前发一条更新。

暂停期间：离开争论空间，不继续发长消息，不删改共同资源，也不在他人面前评理。

## 修复开场

> 昨天你问厨房时，我提高音量并说“你只会挑错”，这是我的具体责任，也会让讨论更难继续。我为这句话道歉，不要求你马上接受。厨房分工本身仍需要重新说清楚；如果你愿意，我们只用十分钟列出“谁、什么时候、做到什么程度”。你也可以选择晚一点或用文字谈。

## 七天观察

只记录三个指标：是否在伤人话前暂停、从暂停到重新联系的时间、同类冲突是否再次升级。第 3 天检查脚本是否可兑现；第 7 天看暂停是否减少伤害，以及厨房规则是否真正变清楚。谈话平静但同类问题继续重复，不算完整解决。

## 不应出现

- 诊断对方为自恋、回避型人格或其他疾病；
- 假定双方权力对等、各打五十大板；
- 要求对方立即原谅或保证关系改善；
- 把无限失联包装成暂停；
- 在出现暴力、胁迫、跟踪或报复风险时继续建议对质、修复或共同沟通。
